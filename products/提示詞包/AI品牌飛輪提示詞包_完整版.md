## 使用說明

### 黃金三角公式

每個提示詞都遵循「黃金三角」結構：

```
角色 + 情境 + 期望
```

- **角色**：告訴 AI 它是誰（專家身份）
- **情境**：提供背景脈絡（你的產業、客戶）
- **期望**：明確說明想要的產出格式

### 使用步驟

1. 選擇適合的模板
2. 將 `[ ]` 中的內容替換成你的資訊
3. 貼到 ChatGPT / Claude / Gemini
4. 根據產出微調優化

## 第一部分：文章撰寫模板（10 個）

### 模板 1：專業知識文章

```
你是一位 [產業] 領域的資深專家，擁有 [年數] 年實戰經驗。

請幫我撰寫一篇關於「[主題]」的專業文章。

目標讀者：[目標客群，如：30-45歲的中小企業主]
文章目的：[建立專業形象 / 解決痛點 / 教育市場]
字數要求：[800-1200] 字

文章結構：
1. 開頭：用一個真實案例或數據吸引注意
2. 問題：點出讀者常見的 3 個痛點
3. 解方：提供具體可執行的建議
4. 結尾：呼籲行動（CTA）

語氣：專業但親切，像朋友在聊天
```

### 模板 2：客戶痛點文章

```
你是一位專門服務 [目標客群] 的 [職業]。

請撰寫一篇文章，主題是：「[目標客群] 最常犯的 [數字] 個錯誤」

寫作要求：
- 每個錯誤都要有具體情境描述
- 說明這個錯誤會造成什麼後果
- 提供正確的做法建議
- 用「你」來稱呼讀者，增加親切感

字數：[800-1000] 字
```

### 模板 3：成功案例故事

```
請幫我將以下客戶案例改寫成一篇動人的故事文章。

客戶背景：[基本資訊]
遇到的問題：[原本的困境]
我們提供的解決方案：[服務內容]
最終成果：[具體數據或改變]

寫作要求：
1. 使用 STAR 架構（情境-任務-行動-結果）
2. 加入客戶的心路歷程變化
3. 結尾帶出服務價值
4. 字數約 600-800 字
```

### 模板 4：趨勢觀點文章

```
你是 [產業] 的趨勢觀察家。

請針對「[趨勢主題]」撰寫一篇觀點文章。

內容要包含：
1. 這個趨勢是什麼？用白話文解釋
2. 為什麼現在很重要？
3. 對 [目標讀者] 有什麼影響？
4. 我的觀點與建議

參考資料：[如有相關新聞或數據可附上]
語氣：有見解但不說教
字數：[600-800] 字
```

### 模板 5：Q&A 整理文章

```
我是一位 [職業]，經常被客戶問到以下問題：

[列出 5-8 個常見問題]

請幫我整理成一篇「常見問題大解答」的文章。

要求：
- 每個問題的回答控制在 100-150 字
- 回答要具體實用，不要打高空
- 適時加入案例說明
- 最後附上一段「還有其他問題？歡迎諮詢」的 CTA
```

### 模板 6：教學步驟文章

```
請幫我寫一篇「如何 [達成某目標]」的教學文章。

目標讀者：[對象]
他們目前的程度：[初學者 / 有基礎 / 進階]

請用以下結構：
1. 為什麼要學這個？（動機）
2. 需要準備什麼？（前置作業）
3. 步驟拆解（每步驟要具體可執行）
4. 常見問題與解決方法
5. 下一步建議

語氣：像老師在教學，耐心且鼓勵
```

### 模板 7：比較分析文章

```
請幫我寫一篇「[選項A] vs [選項B]」的比較分析文章。

讀者背景：[正在考慮選擇的人]
他們的主要考量：[價格 / 效果 / 時間 / 其他]

比較面向：
1. [面向1]
2. [面向2]
3. [面向3]
4. [面向4]

要求：
- 客觀分析，不要偏頗
- 最後給出「什麼情況適合選 A，什麼情況適合選 B」的建議
- 字數約 800-1000 字
```

### 模板 8：清單型文章

```
請幫我寫一篇「[數字] 個 [主題] 的 [技巧/工具/方法]」

例如：「10 個提升業績的 AI 工具」

要求：
- 每個項目要有：名稱、一句話說明、適用情境
- 按照重要性或使用頻率排序
- 開頭說明為什麼整理這份清單
- 結尾提供行動建議
```

### 模板 9：迷思破解文章

```
請幫我寫一篇破解「[主題]」常見迷思的文章。

常見迷思：
1. [迷思1]
2. [迷思2]
3. [迷思3]

每個迷思要包含：
- 為什麼大家會這樣想？
- 真相是什麼？
- 正確觀念是什麼？

語氣：溫和地導正觀念，不要讓讀者覺得被嘲笑
```

### 模板 10：個人故事文章

```
請幫我將以下經歷寫成一篇個人故事文章：

背景：[當時的情況]
轉折點：[發生什麼事]
學到的功課：[領悟]
現在的改變：[成長]

寫作要求：
- 用第一人稱「我」
- 加入當時的情緒和內心對話
- 結尾連結到現在的專業服務
- 讓讀者感受到真誠，而非炫耀
```

## 第二部分：社群貼文模板（5 個）

### 模板 11：金句開場貼文

```
請幫我寫一則 [Facebook/LinkedIn/Instagram] 貼文。

核心訊息：[你想傳達的一個重點]
目標：[引發共鳴 / 建立專業 / 促進互動]

格式要求：
- 第一行用一句金句開場（15字以內）
- 中間 3-5 行說明或故事
- 結尾一個問題邀請留言互動
- 總字數 150-200 字
- 附上 3-5 個相關 hashtag
```

### 模板 12：案例見證貼文

```
請將以下客戶回饋改寫成社群貼文：

客戶原話：「[客戶的回饋內容]」
客戶背景：[簡單描述]
服務內容：[你提供什麼]

貼文要求：
- 開頭用成果數據吸引注意
- 中間說明客戶的改變
- 引用客戶原話增加可信度
- 結尾帶出服務價值，但不要太銷售感
```

### 模板 13：教學小技巧貼文

```
請幫我寫一則「快速學會 [技巧]」的教學貼文。

目標讀者：[對象]
要教的內容：[技巧說明]

格式：
- 開頭點出痛點
- 用 1-2-3 步驟說明方法
- 可以用 emoji 增加易讀性
- 結尾邀請讀者試試看並留言分享
```

### 模板 14：觀點表態貼文

```
我想針對「[議題]」發表看法。

我的立場：[支持/反對/中立但有觀點]
主要論點：[1-2 個核心理由]
想引發的討論：[希望讀者思考什麼]

要求：
- 立場明確但不偏激
- 尊重不同意見
- 結尾用開放式問題邀請討論
```

### 模板 15：日常分享貼文

```
請幫我把以下日常經歷寫成有意義的社群貼文：

今天發生的事：[事件描述]
我的感受或想法：[反思]
可以連結的專業觀點：[如何與你的專業相關]

要求：
- 自然不做作
- 有溫度有人味
- 最後連結到一個小啟發
```

## 第三部分：客戶見證改寫模板（3 個）

### 模板 16：完整見證改寫

```
請將以下客戶回饋改寫成正式的客戶見證。

原始回饋：
「[客戶的原話或對話紀錄]」

客戶資訊：
- 姓名/稱呼：[如何稱呼]
- 職業/公司：[背景]
- 使用服務：[哪項服務]

改寫要求：
1. 保留客戶原意，但讓語句更流暢
2. 加入具體的前後對比
3. 控制在 150-200 字
4. 結尾可加一句推薦語
```

### 模板 17：數據導向見證

```
請根據以下資訊，撰寫一則以數據為主的客戶見證：

客戶類型：[產業/職業]
合作前的狀況：[具體數據，如：月營收 50 萬]
合作後的成果：[具體數據，如：月營收 120 萬]
花費時間：[多久達成]

見證格式：
「在 [時間] 內，從 [原本] 到 [現在]，[一句心得感想]」
```

### 模板 18：故事型見證

```
請將以下客戶歷程改寫成故事型見證：

客戶背景：[原本的處境]
遇到的瓶頸：[卡關的地方]
轉變的契機：[怎麼開始合作]
現在的狀態：[改變後的樣子]
一句話總結：[客戶說的話]

寫作風格：
- 像在說一個朋友的故事
- 有起承轉合
- 結尾讓人想知道「怎麼做到的？」
```

2. **準備你的素材** — 案例、數據、客戶回饋是讓內容有說服力的關鍵
3. **測試不同版本** — 同一個主題可以用不同模板，看哪個效果好
4. **持續優化** — 把效果好的版本存起來，建立你自己的模板庫
