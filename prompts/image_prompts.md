# 圖片生成提示詞模板

## 基本原則

1. **中文字問題**：目前大多數 AI 繪圖工具對中文支援不佳，建議：
   - 生成無文字的底圖
   - 事後用 Canva 或其他工具加上中文字
   - 或使用付費版 Gemini Imagen（支援中文）

2. **解析度考量**：
   - 社群貼文：1080x1080（正方形）或 1200x630（橫式）
   - 部落格封面：1200x630
   - 簡報用圖：1920x1080
   - 印刷品：需要 4K 以上（付費功能）

## 進階技巧

### 請 AI 幫你規劃圖片

```
我正在寫一篇關於 [主題] 的文章，需要在文章中穿插 3 張配圖。

請幫我規劃這 3 張圖片：
1. 說明每張圖的用途（放在文章哪個段落）
2. 提供英文的圖片生成提示詞
3. 建議的風格和色調

文章內容摘要：
[貼上文章摘要]
```

### 風格一致性

```
我需要為我的品牌製作一系列風格一致的圖片。

品牌調性：[專業/親切/創新/...]
主要色系：[顏色]
目標受眾：[描述]

請幫我生成 [數量] 張圖片，主題分別是：
1. [主題一]
2. [主題二]
3. [主題三]

所有圖片要維持一致的視覺風格。
```

### 去除文字的後製指令

如果生成的圖片有不需要的文字：

```
請幫我重新生成這張圖片，但是：
- 完全不要任何文字、字母或符號
- 保持相同的構圖和風格
- 文字原本的位置用背景元素填補
```

## 注意事項

1. **版權問題**：AI 生成的圖片版權規定各平台不同，商用前請確認
2. **品質檢查**：AI 可能生成有瑕疵的圖片（如手指數量錯誤），發布前請檢查
3. **保持一致**：建立品牌視覺規範，讓所有圖片風格統一
4. **備份提示詞**：好用的提示詞要記錄下來，方便日後重複使用
