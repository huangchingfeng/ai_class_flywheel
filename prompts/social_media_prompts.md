# 社群貼文提示詞模板

## Facebook 貼文

### 模板一：知識分享型

```
你是一位擅長經營社群的 [產業] 專家，要把專業知識用親切的方式分享給粉絲。

以下是我想分享的內容重點：
[貼上內容摘要]

請幫我寫一則 Facebook 貼文，要求如下：
- 字數：約 200-300 字
- 開頭用一個引起共鳴的問句或場景
- 中間用條列式列出 3-5 個重點
- 結尾加上互動問題（如：你有類似經驗嗎？）
- 語氣：專業但親切，像在跟朋友聊天
- 不要使用表情符號
- 最後加上 hashtag（3-5 個）
```

### 模板二：情緒共鳴型

```
你是一位善於說故事的社群經營者，擅長用個人經驗引起讀者共鳴。

背景：
[描述一個你或客戶的真實經歷]

請幫我寫一則能引起共鳴的 Facebook 貼文，要求如下：
- 用「我」的第一人稱敘述
- 結構：困境 → 轉折 → 領悟
- 結尾帶出一個金句或反思
- 語氣：真誠、有溫度
- 字數：約 200-400 字
- 不要使用表情符號
```

### 模板三：新聞評論型

```
針對以下新聞，幫我寫一則有觀點的 Facebook 貼文：

新聞內容：
[貼上新聞]

要求：
- 開頭用一句話點出這則新聞的關鍵
- 中間說明這對 [你的目標客群] 有什麼影響
- 結尾給出你的專業建議
- 語氣：專業、有洞察力
- 字數：約 200-300 字
```

## LINE 社群貼文

### 模板一：精華摘要型

```
請把以下文章濃縮成適合 LINE 社群發布的版本：

原文：
[貼上原文]

要求：
- 字數：100 字以內
- 列出 3 個核心重點
- 結尾附上完整文章連結
- 語氣：簡潔有力
- 格式：
  📌 [主題]

  重點摘要：
  1. ...
  2. ...
  3. ...

  👉 完整文章：[連結]
```

### 模板二：活動宣傳型

```
我要在 LINE 社群宣傳以下活動：

活動資訊：
- 活動名稱：[名稱]
- 時間：[時間]
- 地點/方式：[線上/實體]
- 內容重點：[簡述]

請幫我寫一則宣傳文，要求如下：
- 字數：150 字以內
- 清楚列出時間、地點、報名方式
- 強調參加的好處（1-2 點）
- 加上報名連結或 QR Code 說明
```

## 電子報

### 模板一：週報型

```
請幫我寫一封電子報，內容如下：

本週重點：
1. [重點一]
2. [重點二]
3. [重點三]

要求：
- 開頭：親切的問候語
- 結構：
  - 本週精選文章（附連結）
  - 產業動態快報（2-3 則）
  - 實用小技巧（1 個）
  - 下週預告
- 結尾：簽名檔 + CTA（如：有問題歡迎回信）
- 語氣：專業但有溫度，像寫信給朋友
```

### 模板二：單篇推播型

```
請幫我把以下文章改寫成電子報格式：

文章內容：
[貼上文章]

要求：
- 開頭：用一個問題或場景吸引注意
- 中間：文章精華摘要（約 300-500 字）
- 結尾：「想看完整內容？點這裡」+ 連結
- 加上 P.S.（附加訊息，如：下週有免費講座）
```

## Instagram 貼文

### 模板一：懶人包型（輪播圖文字）

```
我要製作一組 IG 懶人包，主題是：
[主題]

內容重點：
[列出 5-7 個重點]

請幫我規劃 7 張輪播圖的文字內容：
- 第 1 張：封面標題（吸睛、有衝擊力）
- 第 2-6 張：每張一個重點（標題 + 2-3 行說明）
- 第 7 張：總結 + CTA（追蹤、收藏、分享）

每張文字要求：
- 標題：10 字以內
- 說明：30 字以內
- 簡潔有力，適合快速閱讀
```

### 模板二：圖文貼文型

```
請幫我寫一則 IG 貼文，主題是：
[主題]

要求：
- 字數：約 100-150 字
- 開頭用一句吸引人的話
- 中間用條列式（3-5 點）
- 結尾加上互動問題
- hashtag：10-15 個（混合大眾標籤和專業標籤）
```

## Podcast 口播稿

### 模板一：將文章轉為口播稿

```
請把以下文章改寫成 Podcast 口播稿：

文章內容：
[貼上文章]

要求：
- 開場白：「下班學 AI Talk，讓你下班輕鬆學 AI，上班信手拈來運用。大家好，我是阿風老師...」
- 語氣：口語化、自然、像在跟朋友聊天
- 適當加入「對不對」「你知道嗎」「舉個例子」等口語詞
- 長度：約 5-8 分鐘（1500-2000 字）
- 結尾：「如果你覺得今天的內容有幫助，歡迎訂閱我們的頻道...」
```

### 模板二：雙人對談稿

```
請把以下內容改寫成雙人對談的 Podcast 稿：

內容主題：
[主題]

重點：
[列出 3-5 個重點]

要求：
- 角色 A：主持人（負責提問、引導）
- 角色 B：專家（負責回答、舉例）
- 對話要自然，有互動感
- 適當加入「哦～原來如此」「這個很重要」等反應
- 長度：約 8-10 分鐘
```

## 快速改寫指令

### 長文變短文
```
請把以下長文濃縮成 [100/200/300] 字的社群貼文：
[貼上長文]

保留核心觀點，語氣要 [專業/親切/有趣]。
```

### 一文多發
```
請把以下文章改寫成三個版本：

原文：
[貼上原文]

1. Facebook 版（300 字，有互動感）
2. LINE 社群版（100 字，重點摘要）
3. 電子報版（500 字，完整但精煉）
```

## 注意事項

1. **了解平台特性**：FB 重互動、LINE 重即時、IG 重視覺、電子報重深度
2. **保持一致性**：同一主題在不同平台的核心訊息要一致
3. **加入個人風格**：AI 產出後，加入你的口頭禪或習慣用語
4. **測試與優化**：觀察哪種風格互動率高，持續調整
