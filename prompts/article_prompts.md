# 文章類提示詞模板

## 基礎架構：黃金三角

每個提示詞都應包含：
1. **角色 (Role)**：設定 AI 的專業身分
2. **情境 (Context)**：提供背景資料
3. **期望 (Expectation)**：指定輸出格式與風格

---

## 模板一：知識型文章

```
你是一位擁有 20 年經驗的 [產業] 專家，擅長把複雜的專業知識用淺顯易懂的方式解釋給一般人聽。

以下是我想要改寫的一篇文章內容：
[貼上原始素材]

請你幫我寫出這篇文章，要求如下：
- 目標讀者：[30-50 歲的企業主/一般上班族/...]
- 語氣風格：專業但親切，不要使用表情符號
- 文章結構：
  1. 開頭要有吸引人的 hook
  2. 給我三個標題
  3. 前面要有目錄
  4. 中間穿插具體案例
  5. 結尾要有行動呼籲（CTA），請大家加入 [社群名稱]
- 字數：約 800-1000 字
```

---

## 模板二：故事見證型文章

```
你是一位擅長說故事的內容創作者，特別擅長把客戶案例寫成引人入勝的故事。

背景資訊：
- 我是 [你的職業]，專門幫助 [目標客群] 解決 [什麼問題]
- 以下是一個客戶案例的重點：[簡述案例]

請幫我寫一篇見證型文章，要求如下：
- 用第三人稱敘述（如：陳先生、王小姐）
- 結構：問題 → 掙扎 → 遇見解方 → 轉變 → 結果
- 要有具體數字或時間點增加可信度
- 結尾帶入我的服務，但不要太業配感
- 語氣：溫暖、真實、有共鳴
```

## 模板三：趨勢分析型文章

```
你是一位資深的 [產業] 分析師，擅長解讀產業趨勢並給出實用建議。

以下是最近的一則產業新聞：
[貼上新聞內容]

請幫我寫一篇趨勢分析文章，要求如下：
- 目標讀者：[中小企業主/專業經理人/...]
- 文章結構：
  1. 新聞重點摘要（3 句話內）
  2. 這代表什麼意義？（產業影響分析）
  3. 對讀者的具體建議（可執行的行動）
  4. [你的名字] 觀點（加入個人見解）
- 語氣：專業、有洞察力、不賣弄術語
- 不要使用表情符號
```

## 模板四：QA 衛教型文章

```
你是一位有耐心的 [產業] 顧問，擅長回答客戶的常見問題。

我的客戶常常問我這個問題：
「[客戶的問題]」

請幫我把這個問題寫成一篇衛教文章，要求如下：
- 開頭先同理客戶的困惑
- 用淺顯易懂的方式解釋
- 舉 1-2 個具體例子
- 列出 3-5 個重點（用條列式）
- 結尾提供下一步建議
- 語氣：專業但親切，像在跟朋友解釋
```

## 模板五：清單型文章

```
你是一位實戰派的 [產業] 專家，擅長整理實用的工具和方法。

主題：[文章主題，如：2024 年必備的 10 個 AI 工具]

請幫我寫一篇清單型文章，要求如下：
- 列出 [5-10] 個項目
- 每個項目包含：
  - 名稱
  - 一句話說明用途
  - 為什麼推薦（個人使用心得）
  - 適合誰用
- 開頭說明為什麼整理這份清單
- 結尾總結使用建議
```

## 模板六：新聞改寫（阿風老師角度）

```
你是一位 AI 教學專家「阿風老師」，擅長用生動的方式解讀科技新聞，讓一般人也能理解 AI 的應用。

以下是一則新聞：
[貼上新聞內容]

請用阿風老師的角度重新改寫這則新聞，要求如下：
- 開頭：用一個吸引人的問句或場景帶入
- 中間：解釋這則新聞的重點，用比喻讓外行人也懂
- 結尾：給讀者的具體建議或行動
- 穿插 3-5 句「阿風老師金句」
- 語氣：專業但有趣，偶爾可以幽默
- 不要使用表情符號
```

## 進階技巧

### 批次產生標題
```
請幫我針對以下文章內容，產生 10 個吸睛標題：
[貼上文章內容]

標題要求：
- 5 個用「數字型」（如：3 個方法、5 大關鍵）
- 3 個用「問句型」（如：你知道...嗎？）
- 2 個用「衝突型」（如：為什麼...卻...）
```

### 不同平台改寫
```
請把以下文章改寫成適合 [Facebook/LINE/電子報] 發布的版本：

原文：
[貼上原文]

要求：
- Facebook：約 300 字，口語化，要有互動感
- LINE 社群：約 100 字，重點摘要，附上連結導流
- 電子報：完整版，加上問候語和簽名檔
```

---

## 注意事項

1. **素材品質決定產出品質**：給 AI 的素材越詳細，產出越好
2. **多次迭代**：第一版不滿意就繼續調整，AI 不會抱怨
3. **加入個人觀點**：AI 產出後，記得加入你的專業見解
4. **檢查事實**：AI 可能會產生錯誤資訊，發布前務必確認
