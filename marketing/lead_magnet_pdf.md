## 使用前你需要知道的事

### 黃金三角公式

每個好的提示詞都包含三個要素：

| 要素 | 說明 | 範例 |
|------|------|------|
| **角色** | AI 要扮演誰 | 「你是 20 年經驗的保險顧問」 |
| **情境** | 背景資料是什麼 | 「以下是一則關於長照的新聞...」 |
| **期望** | 要什麼格式與風格 | 「寫成 300 字 FB 貼文，語氣親切」 |

記住：**你是總編輯，AI 是你的寫手。** 你負責決定方向，AI 負責執行。

## 模板 1：客戶問題變文章

**適用情境**：把客戶常問的問題變成一篇專業文章

```
你是一位 [你的職業]，擁有 [X] 年經驗。

以下是客戶常問我的問題：
「[客戶的問題]」

請幫我把這個回答寫成一篇 Facebook 貼文，要求如下：
- 目標讀者：[你的客群]
- 語氣：專業但親切
- 字數：約 300 字
- 開頭先同理客戶的困惑
- 結尾請大家有問題可以私訊我
```

**使用範例**：
- 保險業務：「買保險要注意什麼？」
- 房仲：「現在是買房的好時機嗎？」
- 財務顧問：「退休金要準備多少才夠？」

## 模板 2：新聞變觀點文

**適用情境**：看到產業新聞，想發表自己的專業觀點

```
你是一位資深的 [產業] 分析師，擅長解讀產業趨勢。

以下是最近的一則產業新聞：
[貼上新聞內容]

請幫我寫一篇趨勢分析文章，要求如下：
- 新聞重點摘要（3 句話內）
- 這代表什麼意義？（產業影響分析）
- 對讀者的具體建議（可執行的行動）
- 加入「阿峰老師觀點」段落
- 語氣：專業、有洞察力
- 字數：約 500 字
```

## 模板 3：見證故事型

**適用情境**：把客戶成功案例寫成動人故事

```
你是一位擅長說故事的內容創作者。

背景資訊：
- 我是 [你的職業]，專門幫助 [目標客群] 解決 [什麼問題]
- 以下是一個客戶案例的重點：[簡述案例]

請幫我寫一篇見證型文章，要求如下：
- 用第三人稱敘述（如：陳先生、王小姐）
- 結構：問題 → 掙扎 → 遇見解方 → 轉變 → 結果
- 要有具體數字或時間點增加可信度
- 結尾自然帶入我的服務
- 語氣：溫暖、真實、有共鳴
- 字數：約 400 字
```

## 模板 4：知識清單型

**適用情境**：整理實用工具、方法、技巧

```
你是一位實戰派的 [產業] 專家。

主題：[X 個你必須知道的 OOO]

請幫我寫一篇清單型文章，要求如下：
- 列出 [5-10] 個項目
- 每個項目包含：
  - 名稱
  - 一句話說明用途
  - 為什麼推薦
  - 適合誰用
- 開頭說明為什麼整理這份清單
- 結尾總結使用建議
- 字數：約 800 字
```

## 模板 5：一文多用（長變短）

**適用情境**：一篇長文拆成多則社群貼文

```
請把以下文章改寫成三個版本：

原文：
[貼上原文]

1. Facebook 版（300 字，有互動感，結尾問一個問題）
2. LINE 社群版（100 字，重點摘要，附上「想看完整版請+1」）
3. IG 限動版（50 字以內，一句金句）
```

## 模板 6：YouTube/Podcast 變文章

**適用情境**：把影音內容轉成文字內容

```
以下是一段影片/Podcast 的逐字稿：
[貼上逐字稿]

請幫我整理成一篇文章，要求如下：
- 萃取 3-5 個核心重點
- 用條列式呈現
- 加入我的專業補充說明
- 結尾加上「想看原始影片請點連結」
- 語氣：專業但易懂
- 字數：約 600 字
```

**取得逐字稿的方法**：
1. 把 YouTube 連結貼到 NotebookLM
2. AI 會自動產生逐字稿和摘要
3. 用這個模板改寫成你的文章

## 模板 7：情緒共鳴型

**適用情境**：用個人經驗引起讀者共鳴

```
你是一位善於說故事的社群經營者。

背景：
[描述一個你的真實經歷，包含：發生什麼事、當時的感受、後來的領悟]

請幫我寫一則能引起共鳴的 Facebook 貼文，要求如下：
- 用「我」的第一人稱敘述
- 結構：困境 → 轉折 → 領悟
- 結尾帶出一個金句
- 語氣：真誠、有溫度
- 字數：約 300 字
```

## 模板 8：週報/電子報型

**適用情境**：固定發送的週報內容

```
請幫我寫一封電子報，內容如下：

本週重點：
1. [重點一]
2. [重點二]
3. [重點三]

要求：
- 開頭：親切的問候語（用「你」不用「您」）
- 結構：
  - 本週精選文章（1 篇，附連結）
  - 產業動態快報（2-3 則，每則 2 句話）
  - 實用小技巧（1 個，可立即使用）
- 結尾：「有問題歡迎直接回信」
- 語氣：專業但有溫度
```

## 模板 9：IG 懶人包規劃

**適用情境**：製作 IG 輪播圖的文字內容

```
我要製作一組 IG 懶人包，主題是：
[主題]

內容重點：
[列出 5-7 個重點]

請幫我規劃 7 張輪播圖的文字內容：
- 第 1 張：封面標題（10 字以內，要吸睛）
- 第 2-6 張：每張一個重點
  - 標題：10 字以內
  - 說明：30 字以內
- 第 7 張：總結 + CTA（追蹤、收藏、分享）

文字要求：簡潔有力，適合快速閱讀
```

## 模板 10：風格切換器

**適用情境**：同一個素材，用不同風格改寫

```
以下是一段內容：
[貼上內容]

請用以下三種風格各寫一個版本，每個約 200 字：

1. 【知識教學型】：像老師在教學生，有條理、有步驟
2. 【故事見證型】：像在說一個真實故事，有情感、有轉折
3. 【趨勢分析型】：像專家在分析產業，有洞察、有預測
```

## 進階技巧：讓 AI 更懂你

### 建立你的「人設說明書」

把這段話存起來，每次對話開頭先貼給 AI：

```
關於我：
- 我是 [職業]，有 [X] 年經驗
- 我的客群是 [描述]
- 我的寫作風格是 [專業/親切/幽默/...]
- 我不喜歡 [表情符號/太正式/太口語/...]
- 我的個人品牌關鍵字：[3-5 個關鍵字]

請用這個風格來協助我寫作。
```

### 持續優化的咒語

當 AI 產出不滿意時，用這些指令調整：

- 「太長了，請濃縮成 200 字」
- 「語氣太正式，請改成像跟朋友聊天」
- 「請加入一個具體的數字或案例」
- 「開頭不夠吸引人，請改成問句開頭」
- 「請加入我的個人觀點，開頭用『我認為...』」
