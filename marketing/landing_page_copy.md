**單元二：先贏一場**
```
30 分鐘產出第一篇文章
- 選定你的第一個主題
- 用 AI 寫出可發布的文章
- 現場完成，帶著成就感離開

產出：一篇可立即發布的社群貼文
```

**單元三：彈藥庫**
```
讓你永遠不缺內容
- 從 YouTube 影片挖掘內容
- 從 Podcast 萃取觀點
- 把新聞變成你的專業文章
- 用 AI 搜尋國內外素材

產出：掌握 5+ 種內容來源
```

**單元四：內容生產線**
```
下指令的技巧
- 黃金三角提示詞公式

- 不同風格切換技巧

產出：10 個可重複使用的提示詞模板
```

**單元五：飛輪系統**
```
從曝光到成交
- 業務飛輪全貌解析

- 演講/拜訪時的導流技巧
```
