# AI 實戰 IP 煉金術：打造個人品牌與自動化流量飛輪

## 課程簡介

協助專業人士（業務、顧問、創業者）利用 AI 解決「沒時間」、「沒靈感」、「不會寫」的痛點，建立個人 IP 權威感，並構建「公域轉私域」的自動化獲客系統。

## 適合對象

- 高客單價業務（房產、保險、金融、B2B 銷售）
- 中小企業主 / 創業者
- 希望建立個人品牌的專業職人（醫師、律師、會計師）

## 課程時數

6 ~ 12 小時（視實作深度而定）

## 課程特色

- **非工程師思維**：不教複雜程式碼，只教聽得懂的商業邏輯與工具應用
- **全流程 SOP 化**：提供課後可直接套用的 Prompt 模板
- **結果導向**：不只學操作，更學如何「省時間」與「賺流量」
