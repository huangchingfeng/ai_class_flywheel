# Part 1：專案背景

## 1.1 關於我們

**阿峰老師（黃敬峰）** — 台灣企業 AI 實戰培訓專家

| 項目 | 數據 |
|------|------|
| 企業合作 | 400+ 家 |
| 培訓學員 | 10,000+ 人 |
| 代表客戶 | 台積電、國泰、富邦、中華電信、ASUS、MODA、遠傳... |
| 教學風格 | 實戰導向，不教理論，只教能用的 |

**核心理念**：
> 「你是機長，AI 是機組人員」
> 「會用、懂用、好用、每天用」

## 1.2 專案緣起

過去阿峰老師主要做企業內訓（B2B），現在想要開拓公開班市場（B2C），針對「個人業務人員」推出 AI 培訓課程。

**為什麼選擇業務人員？**
1. 業務人員的痛點明確：時間寶貴、需要曝光、缺乏內容產出能力
2. 付費能力強：年收百萬的業務，願意投資自己
3. 口碑效應好：業務圈重視人脈推薦，一個成功案例能帶來更多學員
4. 成果可量化：諮詢數、成交率、品牌曝光都能追蹤

## 1.3 商業模式

我們採用「三階段漏斗」模式：

```
說明會（免費）→ 正式課程（$9,000）→ 高階服務（$48,000+）
   │                  │                    │
   ▼                  ▼                    ▼
  撈名單            主要營收            高價值客戶
```

**這份需求的位置**：第一階段「說明會報名頁」— 用來收集潛在客戶名單

# Part 2：目標客群

## 2.1 理想客戶畫像（ICP）

### 一句話描述
> **「超級業務」**：年收入 100 萬以上的高單價業務人員

### 詳細輪廓

| 維度 | 描述 |
|------|------|
| **產業** | 保險、房仲、財務顧問、高端 B2B 銷售、直銷 |
| **年齡** | 30-50 歲 |
| **年收入** | 100 萬以上（時間成本高，願意付費換效率） |
| **工作狀態** | 業績不錯，但每月都在重新開發客戶 |
| **科技程度** | 會用手機、會用 LINE，但對 AI 工具不熟 |
| **學習意願** | 高，知道要進步，但不知道從哪開始 |

### 心理特徵

**他們在想什麼：**
- 「我知道要經營個人品牌，但我真的沒時間」
- 「別人用 AI 好像很厲害，但我試了沒感覺」
- 「我不想學 100 個工具，我只想知道什麼對業務有用」
- 「我願意花錢，但要確定有效」

**他們害怕什麼：**
- 被年輕人取代
- 花時間學了沒用的東西
- 在客戶面前顯得落伍

**他們想要什麼：**
- 客戶主動來找我（不用再追客戶）
- 省時間（把時間花在見客戶，不是做內容）
- 專業形象（讓客戶覺得我很厲害）

## 2.2 為什麼這群人會買單？

| 痛點 | 我們的解法 | 價值 |
|------|-----------|------|
| 沒時間寫內容 | AI 15 分鐘產出一篇文章 | 每週省 5-10 小時 |
| 寫出來很假 | 黃金三角公式讓 AI 寫出你的風格 | 內容像你本人寫的 |
| 不知道寫什麼 | 提供 10+ 種文章模板 | 不用再想題目 |
| 發了沒人看 | 教你飛輪系統，持續累積 | 3 個月後客戶主動來 |

## 2.3 不是我們的客群

| 類型 | 為什麼不適合 |
|------|-------------|
| 剛入行的菜鳥業務 | 還沒有穩定業績，付費能力弱 |
| 不相信 AI 的人 | 說服成本太高 |
| 只想要免費資源的人 | 不會轉換成付費客戶 |
| 想要「一鍵致富」的人 | 我們教的是工具，不是魔法 |

# Part 3：產品架構

## 3.1 完整產品階梯

```
免費 → $699 → $990 → $9,000 → $48,000
  │      │       │        │         │
  ▼      ▼       ▼        ▼         ▼
說明會  線上課  提示詞包  正式課程   密集班
(撈名單) (體驗)  (工具)   (主力)    (深度)
```

### 各產品定位

| 產品 | 價格 | 目的 | 狀態 |
|------|------|------|------|
| **免費說明會** | $0 | 收集名單、建立信任 | 🔴 本次開發 |
| 線上課（三堂） | $699/堂 | 低價體驗、篩選認真的人 | ✅ 課綱完成 |
| AI 提示詞包 | $990 | 提供工具、導流正課 | ✅ 內容完成 |
| 正式課程 | $9,000 | 主要營收來源 | ✅ 已有 |
| 密集班 | $48,000 | 高價值深度服務 | 📋 規劃中 |

## 3.2 說明會的角色

**這場說明會要達成的目標：**

1. **收集名單** — 讓潛在客戶留下聯絡資訊
2. **建立信任** — 讓他們認識阿峰老師、相信這套方法有效
3. **篩選客戶** — 找出真正有需求、有付費能力的人
4. **導購正課** — 說明會結束後，一定比例報名 $9,000 正課

**轉換預期：**
```
報名說明會：100 人
實際出席：60 人（60%）
報名正課：12 人（出席者的 20%）
營收：$108,000
```
