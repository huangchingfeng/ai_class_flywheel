# AI 業務飛輪實戰班：完整課程大綱

> 讓客戶主動找上你的自動化內容系統

---

## 課程定位與目標

**核心目標**：協助業務人員與一人公司老闆，利用 AI 解決「沒時間」、「沒靈感」、「不會寫」的痛點，打造讓客戶主動找上門的內容飛輪系統。

**適合對象**：
- 一人公司老闆
- 高客單價業務（保險、壽險、房產、金融、B2B 銷售）
- 希望建立個人專業形象的職人

**課程時數**：6 小時（基礎版）/ 12 小時（完整版含進階工具）

---

## 第一單元：定位與盤點 - 搞清楚你要寫什麼（1 小時）

**目標**：在動手之前，先釐清你的內容方向與素材來源。

### 1-1 為什麼你需要做內容？

- **信任經濟**：客戶在找你之前，會先 Google 你
- **讓客戶主動來**：從「你去找客戶」變成「客戶來找你」
- **實戰案例**：5 個月產出 143 篇文章的成效數據

### 1-2 你是總編輯，AI 是你的編輯團隊

| 你負責 | AI 負責 |
|--------|---------|
| 決定要寫什麼主題 | 幫你找資料 |
| 審核內容是否正確 | 幫你寫初稿 |
| 加入個人觀點 | 幫你改寫成不同格式 |

### 1-3 內容來源盤點（小組討論）

**使用工具**：[內容來源盤點表](content_source_worksheet.md)

帶領學員思考：我的產業可以從哪裡找素材？

| 來源類型 | 範例 |
|----------|------|
| 新聞與時事 | Google 新聞、產業媒體、國外新聞 |
| 影音內容 | YouTube、Podcast、法說會、線上課程 |
| 演講與培訓 | 公司培訓、公會講座、研討會筆記 |
| 客戶互動 | 常見問題、反對意見、成交故事 |
| 書籍文章 | 專業書籍、雜誌、電子報、部落格 |
| 國際資訊 | 美國/日本/歐洲的產業資訊（資訊差打法） |

**小組討論 20 分鐘**：
1. 你的產業最適合用哪 3 種資料來源？
2. 你的客戶最常問什麼問題？
3. 分享給小組，互相給建議

**單元產出**：完成個人的「內容來源盤點表」

---

## 第二單元：先贏一場 - 30 分鐘產出第一篇文章（1 小時）

**目標**：讓學員在課堂上就完成一篇可發布的文章，建立信心。

### 2-1 選定你的第一個主題

從盤點表中挑選：
- 客戶最常問的一個問題
- 或：最近看到的一則產業新聞

### 2-2 實作：用 AI 寫出你的第一篇文章

**工具**：ChatGPT 或 Gemini（免費版即可）

**步驟**：
1. 把客戶問題或新聞貼給 AI
2. 請 AI 用你的角度改寫成文章
3. 加入你的個人觀點
4. 完成！

**提示詞範本**：
```
你是一位 [你的職業]，擁有 [X] 年經驗。

以下是客戶常問我的問題：
「[客戶的問題]」

請幫我把這個回答寫成一篇 Facebook 貼文，要求如下：
- 目標讀者：[你的客群]
- 語氣：專業但親切
- 字數：約 300 字
- 結尾請大家有問題可以私訊我
```

**現場產出**：一篇可以立刻發布的社群貼文

## 第三單元：彈藥庫 - 讓你永遠不缺內容（1.5 小時）

**目標**：學會從各種來源萃取素材，建立源源不絕的內容庫。

### 3-1 從 YouTube 挖寶

**適用情境**：產業 KOL 的影片、教學內容、法說會、訪談節目

**流程**：
1. 找到一支你覺得不錯的 YouTube 影片
2. 複製影片連結
3. 用 AI 工具取得逐字稿與重點摘要
4. 加入你的專業觀點，改寫成文章

**實作**：選一支 5-10 分鐘的產業影片，萃取 3 個重點

### 3-2 從 Podcast 挖寶

**適用情境**：產業 Podcast、專家訪談、趨勢分析節目

**流程**：
1. 找到一集你覺得有價值的 Podcast
2. 下載音檔或取得連結
3. 用 AI 工具轉成逐字稿
4. 整理成你的觀點文章

**實作**：選一集 Podcast，整理出可發文的 3 個觀點

### 3-3 新聞變觀點文

**適用情境**：產業新聞、時事議題、政策變動

**工具**：Google 新聞、產業媒體網站

**流程**：
1. 搜尋你的產業關鍵字
2. 找到一則有感的新聞
3. 請 AI 加入你的專業觀點改寫

**實作**：找一則新聞，寫成「[你的名字] 觀點」文章

### 3-4 用 AI 搜尋國內外素材

**適用情境**：找靈感、研究趨勢、資訊差打法

**工具**：Perplexity、Gemini Deep Research、Grok

**國內素材**：
- 搜尋產業關鍵字，快速掌握最新動態
- 整合多個來源，省去一個一個網站看的時間

**國外素材（資訊差打法）**：
- 你的產業在哪個國家最先進？（美國？日本？歐洲？）
- 用英文關鍵字搜尋該國的產業資訊
- 翻譯 + 結合台灣現況 = 獨家洞察

**實作**：用 Perplexity 搜尋一個產業問題，整理成文章大綱

### 3-5 錄音與他人內容轉化術

**適用情境**：會議記錄、演講筆記、訪談內容、客戶對話

**工具**：NotebookLM、會議記錄小幫手（如 Otter.ai、Fireflies.ai、好會議）

**應用場景**：

| 來源 | 轉化方式 |
|------|----------|
| 客戶諮詢錄音 | 整理成 FAQ 文章 |
| 公司內訓錄音 | 轉成知識分享文 |
| 演講/研討會 | 萃取重點變貼文 |
| 一對一訪談 | 改寫成見證故事 |

**NotebookLM 使用流程**：
1. 上傳錄音檔、PDF、或貼上 YouTube 連結
2. AI 自動產生摘要與重點
3. 用對話方式追問細節
4. 匯出成你需要的格式

**會議記錄小幫手使用情境**：
- 線上會議自動產生逐字稿
- 重點摘要與待辦事項整理
- 會後快速產出會議紀錄

**實作**：用手機錄一段「回答客戶常見問題」，上傳到 NotebookLM 轉成文章

## 第四單元：內容生產線 - 下指令的技巧（1 小時）

**目標**：學會下精準的指令，讓 AI 寫出「像你」的文章。

### 4-1 黃金三角提示詞公式

| 要素 | 說明 | 範例 |
|------|------|------|
| 角色 | AI 要扮演誰 | 「你是 20 年經驗的保險顧問」 |
| 情境 | 背景資料是什麼 | 「以下是一則關於長照的新聞...」 |
| 期望 | 要什麼格式與風格 | 「寫成 300 字 FB 貼文，語氣親切」 |

### 4-2 一文多用：長文拆成短貼文

**實作**：把第二單元寫的文章，拆成：
- 1 則 FB 完整版（300 字）
- 1 則 LINE 社群版（100 字）
- 3 則金句圖文（每則 30 字）

### 4-3 不同風格切換

同一則素材，改寫成：
- 知識教學型
- 故事見證型
- 趨勢分析型

**實作**：選一種風格，改寫你的文章

## 第五單元：飛輪系統 - 從曝光到成交（1 小時）

**目標**：建立從「發文」到「客戶主動找你」的完整路徑。

### 5-1 業務飛輪全貌

```
┌─────────────────────────────────────────────────┐
│                                                 │
│    曝光 ──→ 圈粉 ──→ 信任 ──→ 諮詢 ──→ 成交    │
│     ↑                                     │     │
│     └─────────── 見證回饋 ←──────────────┘     │
│                                                 │
└─────────────────────────────────────────────────┘
```

| 階段 | 你要做什麼 | 平台 |
|------|-----------|------|
| 曝光 | 發布內容 | FB、IG、部落格 |
| 圈粉 | 導入私域 | LINE 社群、電子報 |
| 信任 | 持續提供價值 | 私域內容、一對一互動 |
| 諮詢 | 客戶主動聯繫 | LINE、電話 |
| 成交 | 提供服務 | - |
| 見證 | 收集回饋變新內容 | 見證文、案例分享 |

### 5-2 LINE 社群經營實戰

- 如何讓人願意加入？
- 社群裡要發什麼內容？
- 引薦階梯：最簡單的引薦方式

**實作**：設計你的 LINE 社群名稱與定位

### 5-3 演講/拜訪時的導流技巧

- 兩個 QR Code 策略
- 用「送簡報」的名義自然導流

## 第六單元：進階武器庫（選修 / 完整版）

> 以下為 12 小時完整版額外內容

### 6-1 文字轉圖片

- **工具**：ChatGPT DALL-E / Gemini Imagen
- **實作**：為文章製作封面圖

### 6-2 文字轉 Podcast

- **工具**：Google AI Studio / Keeper AI
- **實作**：把文章變成 5 分鐘音檔

### 6-3 文字轉簡報

- **工具**：Gamma
- **實作**：把文章變成可下載的 PDF（引流誘餌）

### 6-4 打造個人網站

- 為什麼你需要自己的網站？
- AI 低成本架站概念介紹

## 課後行動清單

學員回家後一週內要完成：

- [ ] 追蹤 3 個資料來源（訂閱、加書籤）
- [ ] 記錄 5 個客戶常問的問題
- [ ] 用 AI 產出 3 篇文章並發布
- [ ] 建立 LINE 社群（或加入講師社群練習）
- [ ] 將成果分享到課程社群

---

## 課程特色

| 特色 | 說明 |
|------|------|
| **第一堂就有產出** | 不是回家再說，現場就寫出第一篇文章 |
| **用業務聽得懂的話** | 不講「流量」，講「客戶主動找你」 |
| **聚焦成交** | 所有內容都連結到「這能幫我成交嗎？」 |
| **工具精簡** | 入門只要會 ChatGPT，不貪多 |
